# AiAgent2025 
- AiAgent2025 is the node js server which communicates to supabase, synthflow API and openai API.

## Local Setup
- Download Nodejs v22
- git clone `git@github.com:SprucesAustralia/AiAgent2025.git`
- cd AiAgent2025
- npm ci
- create .env file
- npm start (This command will start your server)

### .env
```
PORT=*****
SUPABASE_URL=*******
SUPABASE_KEY=*******
SYNTHFLOW_API_KEY=*******
```
